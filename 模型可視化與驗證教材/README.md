# Grad-CAM 教學教材

先前國科會曾提出疑問：「如何確認 Grad-CAM 可視化的關注區域，與人類認知的關鍵區域一致？」為此，我們設計了將 Grad-CAM 與 SAM（Segment Anything Model） 結合的方法。SAM 具備精確分割影像中多個物體的能力，使用者可先透過 SAM 選取並切割出自己關注的高精度區域，再與 Grad-CAM 的可視化結果進行比對，由此評估模型關注區域與人類認知區域的相似程度。
（完整技術細節與範例可參考雲端簡報）
